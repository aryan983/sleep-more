# Can I Sleep More?

186 bus wake-up calculator — Whitchurch Lane → Harrow & Wealdstone.

## Deploy to GitHub Pages

1. Create a new **public** repo on GitHub (e.g. `sleep-more`)
2. Upload all files in this folder to the repo root
3. Go to repo **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Wait ~1 min, then open `https://YOUR-USERNAME.github.io/sleep-more/`

## Add to iPhone Home Screen

1. Open the URL above in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add**

That's it — it'll appear as an app icon and open fullscreen with no browser chrome.
